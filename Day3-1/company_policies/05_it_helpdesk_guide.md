1\. 개요

본 안내는 노트북·계정·사내 시스템 등 IT 문제 발생 시 지원을 받는 절차를 설명한다.

2\. 문의 방법

1.  노트북 고장, 계정 잠금, 소프트웨어 설치 등 IT 문제는 **IT 헬프데스크(helpdesk@hanbit.example
    · 내선 1234)**로 문의한다.

2.  사내 포털 **‘IT 지원 요청’**에서 티켓을 등록할 수도 있다.

3.  운영 시간은 평일 09:00\~18:00이며, 시간 외 긴급 건은 당직 연락처(내선 1239)를 이용한다.

3\. 처리 기준(SLA)

|        |                  |           |
| ------ | ---------------- | --------- |
| **등급** | **예시**           | **목표 응답** |
| 긴급     | 전사 시스템 장애, 보안 사고 | 1시간 이내    |
| 일반     | 노트북 고장, 계정 잠금    | 당일 이내     |
| 요청     | 소프트웨어 설치, 권한 신청  | 2영업일 이내   |
